Custom Instruction Set Computer Simulator (C, x86 Assembly)
Linux • Instruction decode • ALU ops • Register state updates • Debugging with GDB

Built a custom instruction set simulator to model low-level program execution. Debugged silent logic errors where programs ran but produced incorrect output, using GDB to trace execution step-by-step and validate control flow and data-path behavior. Extended the simulator by adding new instructions.

FSMs • Adders • Control logic • RTL testbenches • Waveform analysis

Designed and verified finite state machines and arithmetic/control logic in Verilog. Traced failures back to timing/state-transition issues using RTL testbenches and waveform-based debugging to prove correctness under clocked operation.

Implemented a C++ polynomial library supporting arithmetic operations and evaluation using dynamic memory. Fixed ownership and lifecycle bugs by redesigning constructors, destructors, and operator overloads to enforce safe object-oriented behavior.

Designed, assembled, and tested an LM386 amplifier PCB. After fabrication, diagnosed connectivity and signal issues that weren’t obvious in the schematic, requiring physical inspection and iterative debugging to reach a working build.

Smart Message Terminal (ESP32)
Embedded • UI/UX • Real device behavior • Iteration-driven build

Building an ESP32-based “Smart Message Terminal” focused on clean user experience, reliable behavior, and real-world constraints. Emphasis on debugging, robustness, and making the device feel polished (not just “it works on my desk”).
